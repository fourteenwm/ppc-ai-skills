# Ad Copy Generation Framework

A structured RSA copywriting system with 23 elements, distribution formulas, and the "Four Words to Value" rule for creating high-performing Google Ads responsive search ads.

**The pain point:** Writing 15 headlines and 4 descriptions per RSA without a system leads to repetitive copy, missed angles, and inconsistent quality. This framework gives you a repeatable formula — which headline slots to fill with keywords, social proof, CTAs, and USPs — so every ad has strategic variety built in.

---

## What's Inside

- 23-element copywriting framework organized into 3 tiers (Mandatory, Preferred, Optional)
- Distribution formula for 15 headlines: 3 keyword + 2 social proof + 4 USP + 2 CTA + 1 pun + 3 flexible
- Description formula: 2 with CTA + 2 without
- "Four Words to Value" rule for landing page headline optimization
- Character limit validation rules (30 char headlines, 90 char descriptions)
- Sentiment scoring targets (0.8-1.0 positive)
- Good vs. bad examples for each element

## Positioning Map Templates

### Common Axis Pairs:

**For service businesses:**
- X: Price (Low → High)
- Y: Specialization (Generalist → Specialist)

**For speed-focused businesses:**
- X: Speed (Slow → Fast)
- Y: Quality (Basic → Premium)

**For local businesses:**
- X: Scale (Single location → Multi-location/Chain)
- Y: Personalization (Transactional → Relationship-focused)

**For B2B:**
- X: Company size served (SMB → Enterprise)
- Y: Solution complexity (Simple → Comprehensive)

### How to Plot:
1. Draw the grid
2. Place each competitor based on their positioning
3. Place the client
4. Identify open spaces = opportunities

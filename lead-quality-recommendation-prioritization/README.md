# Lead Quality Recommendation Prioritization

A 3-tier priority system for structuring lead quality recommendations with implementation timelines, verification requirements, and "What Was Ruled Out" documentation.

**The pain point:** Investigations produce a pile of findings, but clients need a clear action plan — what to do today, what to do this week, what to plan for this month. Generic advice like "improve targeting" wastes everyone's time. This skill forces every recommendation to be verified, specific, prioritized, and measurable before it goes into a report.

---

## What's Inside

- 3-tier priority system: Immediate (today), Medium (this week), Long-term (this month)
- Prioritization decision tree for classifying each finding
- Structured recommendation template: Current State, Recommendation, Implementation Steps, Expected Impact, Verification
- "What Was Ruled Out" framework for documenting hypotheses that didn't pan out
- Implementation timeline template (week-by-week roadmap)
- Success metrics and monitoring templates with primary/secondary KPIs
- Pre-flight quality checklist for recommendations
- Real-world examples for each tier (URL exclusions, conversion value rules, offline conversion tracking)
